# Project Images Folder

This folder contains images for your portfolio projects.

## How to Add Your Project Images:

1. Add your project images to this folder
2. Name them using this convention:
   - project-{id}-{image-number}.jpg
   - Example: project-1-1.jpg, project-1-2.jpg, etc.

## Image Requirements:
- Recommended size: 1920x1080 pixels (16:9 aspect ratio)
- Format: JPG, PNG, or WebP
- File size: Under 500KB per image for optimal loading
- Quality: High resolution, professional appearance
